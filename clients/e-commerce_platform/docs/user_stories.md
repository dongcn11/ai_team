# User Stories - E-Commerce Platform

## Feature: Add to Cart

| ID | User Story | Story Points |
|----|------------|--------------|
| US-001 | As a customer, I want to add products to my cart so that I can purchase them later | 3 |
| US-002 | As a customer, I want to view the items in my cart so that I can review before checkout | 2 |
| US-003 | As a customer, I want to update quantities of items in my cart so that I can adjust my order | 3 |
| US-004 | As a customer, I want to remove items from my cart so that I don't purchase unwanted products | 2 |

## Feature: Login

| ID | User Story | Story Points |
|----|------------|--------------|
| US-005 | As a user, I want to log in with my email and password so that I can access my account | 3 |
| US-006 | As a user, I want to reset my password if I forget it so that I can regain access to my account | 5 |
| US-007 | As a user, I want to stay logged in across sessions so that I don't have to log in every time | 3 |
| US-008 | As a user, I want to see clear error messages when login fails so that I know what went wrong | 2 |

## Feature: Payment

| ID | User Story | Story Points |
|----|------------|--------------|
| US-009 | As a customer, I want to pay with credit/debit card so that I can complete my purchase | 8 |
| US-010 | As a customer, I want to save my payment methods for future use so that checkout is faster | 5 |
| US-011 | As a customer, I want to see a confirmation after successful payment so that I know my order went through | 3 |
| US-012 | As a customer, I want to view my payment history so that I can track my purchases | 5 |
| US-013 | As a customer, I want secure payment processing so that my financial data is protected | 8 |
