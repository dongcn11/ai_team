# User Stories

## Feature: Khởi tạo dự án

| # | User Story | Story Points |
|---|------------|--------------|
| 1 | As a user, I want to create a new project so that I can start organizing my audiobook content | 3 |
| 2 | As a user, I want to specify project name and description so that I can identify my project easily | 2 |
| 3 | As a user, I want to select project template so that I can start with predefined structure | 3 |
| 4 | As a user, I want to configure project settings during initialization so that I don't need to configure them later | 2 |
| 5 | As a user, I want to see a confirmation after project creation so that I know the project was created successfully | 1 |
